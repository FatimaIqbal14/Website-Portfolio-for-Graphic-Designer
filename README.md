# Digital Portfolio

A modern, responsive, and animation-rich developer portfolio built with React, Vite, and GSAP. This portfolio showcases projects, skills, and contact information through a clean user experience and smooth interactive animations.

## Features

* Modern responsive design
* Smooth GSAP-powered animations
* Scroll-triggered effects
* Interactive project showcase
* Dedicated About section
* Contact section for inquiries
* Optimized Vite development environment
* Clean component-based architecture

## Tech Stack

* React
* Vite
* JavaScript (ES6+)
* GSAP
* CSS3
* HTML5

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/digital-portfolio.git
```

Navigate into the project directory:

```bash
cd digital-portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
